  Data flow for a single prompt
────────────────────────────────────────────────────────────

Tester types message  (or attaches image / PDF)
         |
         ▼
Flask /chat  ──►  log to usage.db (status=pending)
         |
         ▼
Claude API  ──►  decides which tool to call
         |
         ├──►  search_enclosures.py  ──►  master_web.xlsx
         ├──►  list_by_group.py      ──►  master_web.xlsx
         ├──►  lookup_by_code.py     ──►  master_web.xlsx
         ├──►  scrape_fibox.py       ──►  fibox.com (HTTP)
         ├──►  contacts_lookup.py    ──►  contacts.json
         └──►  read_doc.py           ──►  docs/*.pdf
         |
         ▼
Claude generates final response
(ARCA responses: append ARCA T promo footer)
         |
         ▼
Flask  ──►  update usage.db (status=success/failed)
         |
         ▼
Browser renders markdown response
